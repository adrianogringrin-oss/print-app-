ШАГ 1: ЗАПУСТИТЕ СКРИПТ

В терминале выполните:

```bash
cd "/Users/davidbronto/программа для принтов "
./РАЗВЕРНУТЬ_RAILWAY.sh
```

Скрипт автоматически:
- Проверит все файлы
- Инициализирует Git (если нужно)
- Попросит ввести токены
- Загрузит код в GitHub

═══════════════════════════════════════════════════════════

ШАГ 2: СОЗДАЙТЕ ПРОЕКТ НА RAILWAY

1. Зайдите на https://railway.app
2. Войдите через GitHub
3. Нажмите "New Project"
4. Выберите "Deploy from GitHub repo"
5. Выберите ваш репозиторий
6. Railway автоматически начнет развертывание

═══════════════════════════════════════════════════════════

ШАГ 3: НАСТРОЙТЕ ПЕРЕМЕННЫЕ

В проекте Railway:

1. Откройте ваш сервис
2. Перейдите в "Variables"
3. Добавьте переменные:

   TELEGRAM_BOT_TOKEN = ваш_токен
   OPENAI_API_KEY = ваш_ключ
   PORT = 8080

4. После деплоя получите URL (Settings → Domains → Generate Domain)
5. Добавьте еще одну переменную:

   WEB_APP_URL = https://xxx.railway.app

6. Настройте Telegram Bot:
   - Откройте @BotFather
   - Отправьте /setmenubutton
   - Выберите вашего бота
   - Введите URL: https://xxx.railway.app
