КАК ИСПОЛЬЗОВАТЬ ПРОГРАММУ:

1. Запустите программу (см. выше)

2. Нажмите "📁 Загрузить изображение"
   - Выберите изображение с принтом/надписью

3. Нажмите "✂️ Выбрать область"
   - Зажмите левую кнопку мыши
   - Выделите область с принтом/надписью
   - Отпустите кнопку

4. Нажмите "🔍 Извлечь принт"
   - Программа извлечет выбранную область
   - Удалит фон вокруг принта

5. (Опционально) Нажмите "✨ Улучшить качество"

6. Нажмите "💾 Сохранить PNG"
   - Выберите место для сохранения
   - Результат будет в PNG с прозрачным фоном

ЕСЛИ ПРОГРАММА НЕ ЗАПУСКАЕТСЯ:

1. Проверьте зависимости:
   python3 -c "from PIL import Image; import numpy; import cv2; print('OK')"

2. Если ошибка - установите:
   python3 -m pip install --user --break-system-packages Pillow numpy opencv-python

3. Для tkinter:
   brew install python-tk@3.14

4. Или используйте системный Python:
   /usr/bin/python3 GUI_РАБОТАЮЩАЯ.py
