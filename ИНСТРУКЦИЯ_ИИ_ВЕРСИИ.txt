ЧТО УЛУЧШЕНО:

✓ Сохранение центра - не трогает центральную часть (там принт!)
✓ Адаптивный порог - чем ближе к центру, тем мягче удаление
✓ Защита деталей - текст и линии сохраняются

КАК НАСТРОИТЬ ИИ (опционально):

ШАГ 1: Установите OpenAI
  pip3 install openai

ШАГ 2: Получите API ключ
  1. Зайдите: https://platform.openai.com/api-keys
  2. Создайте ключ
  3. Сохраните: echo "ваш-ключ" > ~/.openai_api_key

ШАГ 3: Готово!
  ИИ функции будут доступны автоматически

БЕЗ ИИ тоже работает - используйте ручной режим!

═══════════════════════════════════════════════════════════

КАК ИСПОЛЬЗОВАТЬ:

РЕЖИМ 1: РУЧНОЙ ВЫБОР (работает всегда)
  1. Запустите: ЗАПУСТИТЬ_С_ИИ.command
  2. Выберите "Ручной выбор области"
  3. Загрузите изображение
  4. Выделите область мышью
  5. Нажмите "Извлечь принт"
  ✓ Фон будет удален умно - края агрессивно, центр мягко

РЕЖИМ 2: ИИ ПО ОПИСАНИЮ (требует OpenAI API)
  1. Запустите: ЗАПУСТИТЬ_С_ИИ.command
  2. Выберите "ИИ по описанию"
  3. Загрузите изображение
  4. Опишите что извлечь:
     - "надпись LINKIN PARK на футболке"
     - "картинка с женщиной"
     - "логотип Олимпиада-80"
     - "принт на майке между красными рамками"
  5. Нажмите "ИИ: Извлечь по описанию"
  6. ИИ автоматически определит область!
  ✓ Затем фон удалится умно

═══════════════════════════════════════════════════════════

КАК РАБОТАЕТ УМНОЕ УДАЛЕНИЕ ФОНА:

1. АНАЛИЗ ТОЛЬКО КРАЕВ
   - Определяет цвет фона по краям (15% от края)
   - Не анализирует центр - там может быть фон принта!

2. АДАПТИВНОЕ УДАЛЕНИЕ
   - Края (0-30%): очень агрессивно (порог 30)
   - Средняя зона (30-70%): постепенный переход (порог 30-60)
   - Центр (70-100%): очень мягко (порог 60) - почти не удаляет!

3. ЗАЩИТА ЦЕНТРА
   - Центральная часть (50%+ от края) НИКОГДА не удаляется автоматически
   - Там остается весь принт, даже если есть фон внутри него
