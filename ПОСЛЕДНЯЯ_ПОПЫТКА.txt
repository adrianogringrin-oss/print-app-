ПРОБЛЕМА:

ПРИЧИНА:
Python использует модули из Homebrew Python (несовместимые)

═══════════════════════════════════════════════════════════

РЕШЕНИЕ - 3 СПОСОБА ЗАПУСКА:

1. ПРЯМОЙ ЗАПУСК С ОЧИЩЕННЫМ ОКРУЖЕНИЕМ ⭐
   --------------------------------------------------------
   Дважды кликните:
     ЗАПУСК_ПРЯМО.command
   
   ИЛИ в терминале:
     ./ЗАПУСК_ПРЯМО.command

2. ЗАПУСК .APP (ОБНОВЛЕН)
   --------------------------------------------------------
   Дважды кликните:
     ИзвлечениеПринтов.app
   
   Теперь использует полностью очищенное окружение

3. РУЧНОЙ ЗАПУСК В ТЕРМИНАЛЕ
   --------------------------------------------------------
   cd "/Users/davidbronto/программа для принтов "
   env -i PATH="/usr/bin:/bin" HOME="$HOME" /usr/bin/python3 GUI_МИНИМАЛЬНАЯ.py

═══════════════════════════════════════════════════════════

ЧТО ИЗМЕНЕНО:

✓ Скрипт запуска использует env -i для полной изоляции
✓ Очищает PYTHONPATH и PYTHONHOME
✓ Использует только системный Python
✓ Не подхватывает модули из Homebrew Python
