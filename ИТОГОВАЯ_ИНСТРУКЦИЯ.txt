КАК ИСПОЛЬЗОВАТЬ ПРОГРАММУ:

ШАГ 1: Запустите программу
  → Дважды кликните на ИзвлечениеПринтов.app
  → Откроется окно с кнопками

ШАГ 2: Загрузите изображение
  → Нажмите "📁 Загрузить изображение"
  → Выберите фото с принтом/надписью на одежде

ШАГ 3: Выберите область
  → Нажмите "✂️ Выбрать область"
  → Зажмите ЛКМ и выделите область с принтом/надписью
  → Отпустите кнопку

ШАГ 4: Извлеките принт
  → Нажмите "🔍 Извлечь принт"
  → Программа обрежет выбранную область
  → Удалит фон вокруг принта

ШАГ 5: (Опционально) Улучшите качество
  → Нажмите "✨ Улучшить качество"

ШАГ 6: Сохраните результат
  → Нажмите "💾 Сохранить PNG"
  → Выберите место для сохранения
  → Готово! PNG с прозрачным фоном

═══════════════════════════════════════════════════════════

ЕСЛИ НЕ ЗАПУСКАЕТСЯ:

1. Убедитесь, что Pillow установлен:
   /usr/bin/python3 -m pip install --user --break-system-packages Pillow

2. Попробуйте прямой запуск:
   /usr/bin/python3 GUI_ТОЛЬКО_PIL.py

3. Проверьте ошибки в файле:
   ~/Desktop/gui_app_errors.log
