  🚀 РАЗВЕРТЫВАНИЕ НА RAILWAY - ПОШАГОВАЯ ИНСТРУКЦИЯ
═══════════════════════════════════════════════════════════

Следуйте этим шагам для развертывания приложения на Railway.

═══════════════════════════════════════════════════════════

ШАГ 1: ПОДГОТОВКА

✅ Убедитесь, что у вас есть:
   - Telegram Bot Token (от @BotFather)
   - OpenAI API Key (от platform.openai.com)
   - GitHub аккаунт (если еще нет - создайте на github.com)

═══════════════════════════════════════════════════════════

ШАГ 2: СОЗДАНИЕ РЕПОЗИТОРИЯ НА GITHUB

1. Зайдите на github.com
2. Нажмите "New repository"
3. Назовите репозиторий (например: telegram-print-extractor)
4. Выберите "Public" или "Private"
5. НЕ добавляйте README, .gitignore или лицензию
6. Нажмите "Create repository"

═══════════════════════════════════════════════════════════

ШАГ 3: ЗАГРУЗКА КОДА В GITHUB

Откройте терминал в папке проекта и выполните:

```bash
cd "/Users/davidbronto/программа для принтов "

# Инициализируем git (если еще не инициализирован)
git init

# Добавляем все файлы
git add .

# Делаем первый коммит
git commit -m "Initial commit: Telegram Print Extractor"

# Добавляем удаленный репозиторий (замените YOUR_USERNAME и YOUR_REPO)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# Отправляем код
git branch -M main
git push -u origin main
```

Замените YOUR_USERNAME и YOUR_REPO на ваши данные.

═══════════════════════════════════════════════════════════

ШАГ 4: СОЗДАНИЕ ПРОЕКТА НА RAILWAY

1. Зайдите на https://railway.app
2. Нажмите "Login" и войдите через GitHub
3. Нажмите "New Project"
4. Выберите "Deploy from GitHub repo"
5. Выберите ваш репозиторий
6. Railway автоматически начнет развертывание

═══════════════════════════════════════════════════════════

ШАГ 5: НАСТРОЙКА ПЕРЕМЕННЫХ ОКРУЖЕНИЯ

1. В проекте Railway нажмите на ваш сервис
2. Перейдите в раздел "Variables"
3. Нажмите "New Variable" и добавьте:

   Переменная: TELEGRAM_BOT_TOKEN
   Значение: ваш_токен_бота

   Переменная: OPENAI_API_KEY
   Значение: ваш_ключ_openai

   Переменная: PORT
   Значение: 8080

4. Сохраните все переменные

═══════════════════════════════════════════════════════════

ШАГ 6: ПОЛУЧЕНИЕ URL

1. В проекте Railway перейдите в "Settings"
2. Найдите раздел "Domains"
3. Нажмите "Generate Domain"
4. Скопируйте полученный URL (например: https://xxx.railway.app)

═══════════════════════════════════════════════════════════

ШАГ 7: ОБНОВЛЕНИЕ WEB_APP_URL

1. В разделе "Variables" добавьте еще одну переменную:

   Переменная: WEB_APP_URL
   Значение: https://xxx.railway.app

   (Замените xxx.railway.app на ваш реальный URL)

2. Сохраните переменную

═══════════════════════════════════════════════════════════

ШАГ 8: НАСТРОЙКА TELEGRAM BOT

1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте команду: /setmenubutton
4. Выберите вашего бота из списка
5. Введите URL вашего приложения: https://xxx.railway.app
6. Готово!

═══════════════════════════════════════════════════════════

ШАГ 9: ПРОВЕРКА

1. Откройте вашего бота в Telegram
2. Отправьте команду: /start
3. Нажмите кнопку "Открыть приложение"
4. Загрузите изображение и протестируйте

═══════════════════════════════════════════════════════════

🔍 РЕШЕНИЕ ПРОБЛЕМ:

❌ Приложение не запускается:
   - Проверьте логи в Railway (вкладка "Deployments" → "View Logs")
   - Убедитесь, что все переменные окружения установлены
   - Проверьте, что файлы TELEGRAM_BOT_SERVER.py и TELEGRAM_WEBAPP_SERVER.py есть в репозитории

❌ Бот не отвечает:
   - Проверьте TELEGRAM_BOT_TOKEN
   - Проверьте логи бота в Railway

❌ ИИ не работает:
   - Проверьте OPENAI_API_KEY
   - Проверьте баланс на OpenAI аккаунте

❌ Web App не открывается:
   - Проверьте WEB_APP_URL (должен быть HTTPS)
   - Убедитесь, что веб-сервер запущен (проверьте логи)
