═══════════════════════════════════════════════════════════

СПОСОБ 1: АВТОМАТИЧЕСКИЙ (РЕКОМЕНДУЕТСЯ) ⭐⭐⭐
═══════════════════════════════════════════════════════════

Запустите скрипт настройки:

  ./НАСТРОИТЬ_ВСЕ.sh

Скрипт спросит API ключ и все настроит автоматически!

═══════════════════════════════════════════════════════════

СПОСОБ 2: ИНТЕРАКТИВНЫЙ СКРИПТ
═══════════════════════════════════════════════════════════

Запустите Python скрипт:

  python3 ДОБАВИТЬ_КЛЮЧ.py

Скрипт спросит ключ (не будет показывать на экране) и сохранит его.

═══════════════════════════════════════════════════════════

СПОСОБ 3: РУЧНАЯ УСТАНОВКА
═══════════════════════════════════════════════════════════

1. Создайте файл с ключом:
   echo "ваш-api-ключ-здесь" > ~/.openai_api_key

2. Установите права доступа:
   chmod 600 ~/.openai_api_key

3. Готово!

═══════════════════════════════════════════════════════════

Можете прислать ключ здесь, и я его установлю командой:

  python3 ДОБАВИТЬ_КЛЮЧ.py ваш-ключ-здесь

НО! Лучше использовать один из способов выше - это безопаснее.

═══════════════════════════════════════════════════════════

ГДЕ ВЗЯТЬ API КЛЮЧ:
═══════════════════════════════════════════════════════════

1. Зайдите: https://platform.openai.com/api-keys
2. Войдите в аккаунт (или зарегистрируйтесь)
3. Нажмите "Create new secret key"
4. Скопируйте ключ (начинается с "sk-...")
5. Вставьте в скрипт настройки

ВАЖНО: Ключ показывается только ОДИН раз! Сохраните его.

═══════════════════════════════════════════════════════════

ПРОВЕРКА РАБОТЫ:
═══════════════════════════════════════════════════════════

После установки ключа проверьте:

  python3 ДОБАВИТЬ_КЛЮЧ.py

(запустите без аргументов - он проверит что все настроено)

Или проверьте вручную:

  python3 -c "import os; key = open(os.path.expanduser('~/.openai_api_key')).read().strip(); print('Ключ:', key[:10] + '...')"
