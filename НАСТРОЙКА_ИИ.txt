ПРОБЛЕМА РЕШЕНА!

Добавлена поддержка ИИ для точного понимания что именно нужно извлечь.
Теперь можно давать конкретное задание текстом!

═══════════════════════════════════════════════════════════

КАК НАСТРОИТЬ ИИ:

ШАГ 1: Установите OpenAI библиотеку
  pip3 install openai

ШАГ 2: Получите API ключ OpenAI
  1. Зайдите на: https://platform.openai.com/api-keys
  2. Создайте новый API ключ
  3. Скопируйте ключ

ШАГ 3: Сохраните ключ
  Способ 1 (рекомендуется):
    echo "ваш-api-ключ" > ~/.openai_api_key
  
  Способ 2:
    export OPENAI_API_KEY="ваш-api-ключ"
    (добавьте в ~/.zshrc для постоянного использования)

ШАГ 4: Запустите программу
  ./ЗАПУСТИТЬ_С_ИИ.command

═══════════════════════════════════════════════════════════

КАК ИСПОЛЬЗОВАТЬ ИИ:

1. Запустите: ЗАПУСТИТЬ_С_ИИ.command

2. В браузере выберите: "ИИ по описанию"

3. Загрузите изображение с принтом

4. В поле ввода опишите что нужно извлечь:
   - "надпись LINKIN PARK на футболке"
   - "картинка с женщиной и глобусом"
   - "логотип Олимпиада-80"
   - "принт на майке между красными рамками"
   И т.д.

5. Нажмите "🔍 ИИ: Извлечь по описанию"

6. ИИ автоматически определит область и извлечет!

═══════════════════════════════════════════════════════════

ПРЕИМУЩЕСТВА ИИ:

✓ Понимает контекст - различает фон майки и фон принта
✓ Точное определение области по описанию
✓ Не нужно вручную выделять область
✓ Работает со сложными изображениями
✓ Может извлечь несколько объектов по описанию

═══════════════════════════════════════════════════════════

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:

"надпись LINKIN PARK" - извлечет только текст
"картинка на футболке" - извлечет графику
"логотип в красной рамке" - извлечет логотип
"принт между складками" - точно определит область

═══════════════════════════════════════════════════════════

ЕСЛИ ИИ НЕ РАБОТАЕТ:

1. Проверьте установку:
   python3 -c "import openai; print('OK')"

2. Проверьте ключ:
   cat ~/.openai_api_key

3. Используйте ручной режим - он работает всегда!
