═══════════════════════════════════════════════════════════

ПРОБЛЕМА РЕШЕНА!

Создана ВЕБ-ВЕРСИЯ, которая:
✓ Работает в браузере (нет нативных библиотек = нет крашей!)
✓ Позволяет выделить область мышью
✓ Не требует ввода координат вручную
✓ Простой и понятный интерфейс

═══════════════════════════════════════════════════════════

КАК ЗАПУСТИТЬ:

Дважды кликните на:
  ЗАПУСТИТЬ_ВЕБ.command

ИЛИ в терминале:
  python3 ВЕБ_ВЕРСИЯ.py

После запуска:
✓ Откроется браузер автоматически
✓ Интерфейс загрузится на http://localhost:8000

═══════════════════════════════════════════════════════════

КАК ИСПОЛЬЗОВАТЬ:

ШАГ 1: Загрузите изображение
  → Нажмите кнопку "📁 Загрузить изображение"
  → Выберите фото с принтом/надписью

ШАГ 2: Выделите область мышью
  → Изображение появится на экране
  → Зажмите ЛЕВУЮ КНОПКУ МЫШИ
  → Выделите область с принтом/надписью
  → Отпустите кнопку мыши
  → Нажмите кнопку "🔍 Извлечь принт"

ШАГ 3: Сохраните результат
  → Результат появится внизу
  → Нажмите "💾 Скачать PNG"
  → Файл скачается автоматически!

═══════════════════════════════════════════════════════════

ПРЕИМУЩЕСТВА ВЕБ-ВЕРСИИ:

✓ НЕТ КРАШЕЙ - работает в браузере, не использует нативные библиотеки
✓ Удобный интерфейс - выделение мышью
✓ Не нужно вводить координаты вручную
✓ Работает на всех системах
✓ Простой и понятный

═══════════════════════════════════════════════════════════

ПРИМЕР ИСПОЛЬЗОВАНИЯ:

1. Запустите: ЗАПУСТИТЬ_ВЕБ.command
2. Браузер откроется автоматически
3. Нажмите "Загрузить изображение"
4. Выберите фото с надписью на футболке
5. Выделите мышью область с надписью
6. Нажмите "Извлечь принт"
7. Нажмите "Скачать PNG"
8. Готово! PNG с надписью без фона!

═══════════════════════════════════════════════════════════

ОСОБЕННОСТИ:

- Работает через локальный веб-сервер
- Интерфейс открывается в браузере
- Не требует установки дополнительных библиотек GUI
- Все функции доступны через веб-интерфейс

═══════════════════════════════════════════════════════════

ЕСЛИ НЕ ЗАПУСКАЕТСЯ:

  /usr/bin/python3 --version

Установите Pillow:
  python3 -m pip install --user Pillow
