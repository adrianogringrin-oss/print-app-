СПОСОБ 1: АВТОМАТИЧЕСКАЯ НАСТРОЙКА (РЕКОМЕНДУЕТСЯ) ⭐⭐⭐
═══════════════════════════════════════════════════════════

1. Запустите скрипт настройки:
   ./НАСТРОИТЬ_ВСЕ.sh
   
   Скрипт автоматически:
   ✓ Установит все библиотеки (Pillow, OpenAI)
   ✓ Настроит API ключ (спросит у вас)
   ✓ Проверит что все работает
   ✓ Готово к использованию!

2. Готово! Запускайте:
   ./ЗАПУСТИТЬ_С_ИИ.command

═══════════════════════════════════════════════════════════

СПОСОБ 2: РУЧНАЯ УСТАНОВКА КЛЮЧА
═══════════════════════════════════════════════════════════

Если хотите установить ключ вручную:

1. Установите библиотеки:
   python3 -m pip install --user --break-system-packages Pillow openai

2. Установите ключ:
   echo "ваш-api-ключ-здесь" > ~/.openai_api_key
   chmod 600 ~/.openai_api_key

   ИЛИ используйте скрипт:
   ./УСТАНОВИТЬ_КЛЮЧ.sh ваш-api-ключ

3. Готово!

═══════════════════════════════════════════════════════════

ГДЕ ВЗЯТЬ API КЛЮЧ:
═══════════════════════════════════════════════════════════

1. Зайдите на: https://platform.openai.com/api-keys
2. Войдите в аккаунт (или создайте новый)
3. Нажмите "Create new secret key"
4. Скопируйте ключ (он начинается с "sk-...")
5. Сохраните ключ (он показывается только один раз!)

═══════════════════════════════════════════════════════════

ВАЖНО - БЕЗОПАСНОСТЬ:
═══════════════════════════════════════════════════════════

⚠ НЕ ПУБЛИКУЙТЕ API КЛЮЧ в открытом доступе!
⚠ Ключ сохраняется в ~/.openai_api_key (только для вас)
⚠ Файл защищен (chmod 600) - только вы можете читать
